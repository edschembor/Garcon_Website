FAQs

What is Garcon?

Garcon is an application currently in development for Android and web platforms. The app allows users to create "tasks" with prices and deadlines which other users can then complete in return for the task price. Garcon essentially allows for users to have personal "butlers" when needed and allows others to make money as a "deliverer" without having to be employed by a single organization.

Is a Venmo account necessary for using Garcon?

Yes. See above.

How does the payment system work?

When a user creates a task, the price of the task is taken from their Venmo account and placed in a temporary holding place. If the user has insufficient funds, the task will not be created. Then, after a task is completed and both users have signaled that completion has occured, the money is moved from the temporary bin to the account of the user who completed the task.

What if I complete a task and the person who created it doesn't say I completed it?

First, a user will never be able to create a task if they have insufficient funds. Second, the rating system prevent bad users (see below). Our chat system allows users to make sure they are on the same page regarding the details of the task and its completion. If issues are still had, contact our support team who will work to resolve them.

How does the rating system work?

Each user has a rating for creating tasks and a rating for completing tasks. After a task is completed, the two users rate each other. If a users rating for either falls below 4, the user's account is suspended. If a users rating for either falls below 3, the user's account is banned.
